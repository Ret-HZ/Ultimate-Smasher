# Ultimate Smasher
A Super Smash Bros. Ultimate (Nintendo Switch) save editor written in Python

Features
=====
With Ultimate Smasher you can edit the amounts of:

* Gold
* SP
* Hammers
* Tickets
* Snacks (S, M, L)


It also allows to:
* Unlock/Lock characters
* Get all Cores
* Get all stickers
* Get all spirits
* Unlock all milestones
* Unlock Mii clothes
* Unlock stores
* Unlock Exploration
* Unlock gym
* Unlock dojos
* Unlock music
* Get a full sized Spirit Board
